This Project was submitted in the software engineering course in the American University of Beirut

This Project is a demo of a real web application of a Sport's center where the clients can view courses offered by the Sport's Center, 
register as members,etc..

For the code:

Front end:

JavaScript, HTML/CSS

Backend:
PHP/MYSQL

This project includes backend and front end form validation

Directories:

sign: code for the member to sign Up

members: includes the activities that members can register, the code of the forums that members can ask and answer questions about the Sport's Center

job: Coaches can upload their CVs to the website
